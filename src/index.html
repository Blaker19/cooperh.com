3D Modeling

Bloomoceros

This project was for a class I took in high school my senior year. I used Zbrush to sculpt and polypaint the model, and then I rendered it out in separate layers so I could have more control when I took it into photoshop to create the final image. Year: 2023

Animation

Magmaceros

Submission for Punisher’s “Boss Fight Challenge.” I used the Bloomoceros model I created in Zbrush, re-polypainted it, used the quad-draw tool in maya to retopologize the model to be able to be animated inside of Maya. Animated, lit, and rendered inside of Maya. The character running is from Mixamo. Year: 2023
